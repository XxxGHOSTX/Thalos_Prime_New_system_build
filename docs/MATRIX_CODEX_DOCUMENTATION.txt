================================================================================
                    MATRIX CODEX DOCUMENTATION
================================================================================

PURPOSE:
Matrix Codex provides advanced matrix operations and codex storage.

COMPONENTS:
- MatrixCodex class: Matrix operations
- MatrixCodexIntegration: Integration layer

USAGE:
  from matrix_codex import MatrixCodex
  
  m1 = MatrixCodex.from_list([[1, 2], [3, 4]])
  m2 = MatrixCodex.from_list([[5, 6], [7, 8]])
  result = m1.multiply(m2)

OPERATIONS:
- multiply: Matrix multiplication
- transpose: Matrix transpose
- add_to_codex: Store metadata
- get_from_codex: Retrieve metadata

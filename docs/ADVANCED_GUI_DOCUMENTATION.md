# THALOS Prime Advanced GUI Documentation

## Overview

The advanced GUI provides enhanced interaction capabilities for THALOS Prime.

## Features

### Console Mode
- Available when tkinter is not installed
- Full command support
- History tracking

### Commands
- `/help` - Show help
- `/settings` - Show settings
- `/history` - Show conversation history
- `/clear` - Clear history
- `/set <key> <value>` - Set configuration
- `/quit` - Exit

### Configuration
- `theme` - UI theme (dark/light)
- `font_size` - Text size
- `show_metrics` - Display confidence scores

## Running

```bash
python thalos_prime_advanced_gui.py
```
